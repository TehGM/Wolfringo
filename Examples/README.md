# Example projects
Projects in this directory show examples of basic Wolfringo usage:

- [SimplePingBot](SimplePingBot) - Most basic usage of Wolfringo library, without commands or [.NET Generic Host](https://docs.microsoft.com/en-gb/aspnet/core/fundamentals/host/generic-host?view=aspnetcore-3.0).
- [HostedPingBot](HostedPingBot) - Most basic usage of Wolfringo library when used with [.NET Generic Host](https://docs.microsoft.com/en-gb/aspnet/core/fundamentals/host/generic-host?view=aspnetcore-3.0), without commands.
- [SimpleCommandsBot](SimpleCommandsBot) - Example of configuring and using Commands System, without [.NET Generic Host](https://docs.microsoft.com/en-gb/aspnet/core/fundamentals/host/generic-host?view=aspnetcore-3.0).
- [HostedCommandsBot](HostedCommandsBot) - Example of configuring and using Commands System with [.NET Generic Host](https://docs.microsoft.com/en-gb/aspnet/core/fundamentals/host/generic-host?view=aspnetcore-3.0).
