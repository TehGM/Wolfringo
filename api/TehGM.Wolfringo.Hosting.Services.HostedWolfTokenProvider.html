    Class HostedWolfTokenProvider

    A WOLF token provider that supports both options as well as registered provider.

    Inheritance
    System.Object
    HostedWolfTokenProvider

    Namespace: TehGM.Wolfringo.Hosting.Services
    Assembly: Wolfringo.Hosting.dll
    Syntax
    public class HostedWolfTokenProvider : IWolfTokenProvider
    Remarks

    This token provider reads options to get pre-configured token. If options do not provide the token, a IWolfTokenProvider instance registered with DI will be used.

    Value generated by IWolfTokenProvider is cached internally on a per-instance basis, so all reconnections use the same token.
